# 📋 更新日志 — v2.1

## 🎮 键盘导航与可配置控制

### WASD 键盘导航
- `W` / `A` / `S` / `D` 键持续按住可平移画布（无选中模块时）或微调选中模块位置（有选中模块时）
- 采用 `requestAnimationFrame` 循环实现流畅的逐帧移动

### 可配置控制参数（保存至设计 JSON）
- **移动速度滑条**（5–40 px/帧，默认 20）：控制 WASD 移动速度
- **缩放按键重绑定**：点击按键按钮后按下任意键完成重绑定，默认 `[` 放大、`]` 缩小
- **缩放步进滑条**（3%–10%/帧，默认 5%）：控制每帧缩放幅度
- **说明面板键**（默认 `h`）：按下切换打开/关闭快捷键说明弹窗，可重绑定
- **适应视图键**（默认 `y`）：按下立即执行适应视图，可重绑定
- **侧边栏收起键**（默认 `c`）：按下切换侧边栏收起/展开，可重绑定
- **模块树全屏键**（默认 `x`）：按下切换模块层次树全屏/取消全屏，可重绑定
- **应用全屏键**（默认 `z`）：按下切换浏览器全屏模式（与工具栏⚗ 按钮功能一致），可重绑定
- 所有参数按设计保存至服务端 JSON，重新打开设计后自动恢复

### 快捷键说明面板
- 工具栏新增 **说明** 按钮（视图操作区），或按 `h` 键（可重绑定）切换打开/关闭弹窗
- 弹窗分两区：**⚙ 可自定义按键**（含速度/步进滑条 + 全部可重绑定键）；**📋 快捷键一览**（全量参考表）
- 弹窗内容支持滚动，滚动条样式与整体 UI 一致，不遮挡文字（`scrollbar-gutter: stable`）
- 点击弹窗背景或关闭按钮可关闭

## ⚙️ 模块设置面板增强

### 端口面板（端口 Tab）
- 端口列表采用三列布局：端口名 | 复制按钮 | 位宽
- Chisel 生成的 RTL 中带数字后缀的同名端口（如 `io_a_0`、`io_a_1`）自动分组折叠显示，点击展开
- 搜索框：实时过滤端口列表
- 分组计数展示（如"2 组 / 8 个"）

### 寄存器面板（寄存器 Tab）
- 与端口面板布局一致：三列 + 分组折叠 + 搜索框
- Tab 按钮显示分组/寄存器计数
- 寄存器定义来自 Verilog `reg` 声明，支持数组寄存器识别

### 复制按钮
- 每个端口/寄存器名旁有 **复制** 按钮，点击后显示 ✓ 反馈 500 ms 后恢复

## 📂 侧边栏改进

### 折叠/展开状态持久化
- 侧边栏展开宽度、折叠状态均保存至设计 JSON（字段 `sidebar_ui`）
- 折叠后展开能恢复上次手动拖拽的宽度

### 调整大小手柄修复
- 修复了 JSON 恢复后调整手柄位置不正确的问题（使用 `.resizing` 禁用过渡 + 强制 reflow）
- 修复了初始加载后手柄不出现的边缘情况（双 rAF + window load 监听）

### 滚动条内边距
- 端口/寄存器 Tab 面板增加 `padding-right:8px`，防止滚动条遮挡内容

## 🏗️ 工程变更

### 后端
- `save_state` 接口新增 `canvas_controls` 字段支持（与 `sidebar_ui` 同级保存）

### 前端
- `state.canvasControls` 对象统一管理所有控制参数
- `_saveCanvasControls()` 辅助函数：修改参数后立即触发服务端同步
- `escHtml(s)` 辅助函数：弹窗 HTML 拼接时防止 XSS
- 模态弹窗内容改为由 `showShortcutHelp()` 动态生成，HTML 模板仅保留空壳 `div`
