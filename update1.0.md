## 实现记录 (v1.0 更新)

### ✅ 1. JSON保存源文件 + 刷新按钮
- **verilog_parser.py**: `analyze_and_save()` 现在额外收集所有解析文件路径，保存为 `source_files` 列表
- **app.py**: 新增 `/api/refresh` 端点，根据设计名重新分析源文件
- **index.html**: 全屏按钮左侧新增 🔄 刷新按钮
- **app.js**: 新增 `refreshDesign()` 函数，调用 `/api/refresh` 后自动重新加载设计

### ✅ 2. 线条高亮在拖拽后不再消失
- 新增 `state.justFinishedDrag` 守卫标志
- 模块拖拽结束、调整大小结束、拐点拖拽结束、平移结束后设置该标志
- SVG 背景点击取消选中时，如果刚结束拖拽操作则跳过取消
- `selectedWireKey` 在 `renderCanvas()` 重建SVG后会自动重新应用

### ✅ 3. 框选工具
- **Shift+鼠标拖拽** 在空白区域开始框选
- 橡皮筋矩形实时显示选区范围
- 松开后自动选中范围内的模块和线路拐点
- 选中的模块添加蓝色虚线边框（`.box-selected`）
- 选区边框可拖拽，一起移动所有选中的模块和拐点
- 选区右上角有红色 ✕ 关闭按钮，点击清除选区
- 点击空白区域不会清除框选（必须用关闭按钮）

### ✅ 4. 模块层次单击直接跳转（修订）
- 单击侧边栏模块树节点时分三种情况：
  1. **模块在当前视图可见** → 直接平移居中并高亮
  2. **模块有子实例（可展开）** → 切换到该模块的内部视图并适应窗口
  3. **叶子模块（无子实例）** → 找到包含该模块的上级模块，切换到上级视图，然后自动平移高亮到该模块位置
- 新增 `findParentModule(designName, childModName)` 辅助函数，遍历所有模块的 instances 找到父模块
- 双击树节点仅展开/折叠侧边栏树（不影响画布视图）

### ✅ 5. 画布背景颜色 + 导出（修订）
- **画布背景颜色**：侧边栏导出区新增颜色选择器 + 「透明」按钮
  - 选择颜色 → 立即更新画布背景并持久化到 localStorage
  - 导出时使用该颜色作为背景（SVG 添加 `<rect>` 背景层，PNG 填充 Canvas 背景）
  - 选择「透明」→ 画布显示正常（默认深色），但导出 PNG/SVG 时背景透明
- **模块颜色、重命名**：右键点击模块 → ⚙️ 设置弹窗（同上）
- **注释（修订）**：注释不再显示在模块框上，改为点击模块时以浮动弹窗形式展示
  - 支持 **Markdown 渲染**（引入 marked.js）：标题、加粗/斜体、代码块、列表、引用等
  - 支持 **导入 .md 文件**：设置弹窗中点击「📂 导入 .md 文件」按钮可直接导入本地 md 文件作为注释内容
  - 弹窗位置自动避免超出视口边界，点击画布空白区域关闭

### ✅ 6. 长文本自动处理
- 模块标题截断阈值提升到 36 字符（之前28），截断时显示省略号
- 截断的标题添加 SVG `<title>` tooltip，悬停可查看完整名称
- 模块宽度上限设为 500px，防止超长名称撑破布局
- 信息面板添加 `word-break: break-word` CSS，长线路名不再溢出

### ✅ 7. 撤销/重做
- **Ctrl+Z** 撤销，**Ctrl+Y** 或 **Ctrl+Shift+Z** 重做
- 在每次修改操作（模块拖拽、调整大小、拐点拖拽/添加/删除、框选移动）之前自动保存快照
- 快照内容：`layoutOverrides`（模块位置/大小）和 `wireWaypoints`（线路拐点）
- 撤销栈最多保留 50 条历史记录
- 在输入框/文本框中不拦截快捷键
